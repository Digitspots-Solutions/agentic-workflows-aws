Executive Summary
The Deployment Guide ensures that the solution is reproducible. It provides step-by-step instructions for any engineer to build and deploy the LAUTECH environment from scratch using the provided code artifacts.

1. Guide Overview
1.1 Contents
The guide (located in `docs/PRODUCTION.md`) covers:
1.  **Prerequisites**: CLI tools (AWS, Docker, Python) and Access Keys.
2.  **Configuration**: Environment variables for Database and Bedrock settings.
3.  **Execution**: The exact sequence of scripts to run (`setup_rds.py` -> `deploy_ecs.py`).
4.  **Verification**: How to test the deployment once complete.
